Fox and Geese

Start a new game
Fox and Geese is a classic assymetric board game in the tafl family. The animals only move along the lines drawn on the board. Some squares allow diagonal moves but others do not. The fox can move one square or jump a goose in any allowed direction. The fox can jump only one goose at a time, but can continue jumping geese in its turn as long as a jump is possible. The geese can only move one square. Since the fox can only jump over a single goose, a goose with another goose behind it is safe from jumping. The geese win by trapping the fox. The fox wins when there are four or fewer geese (and so the fox can no longer be trapped).
